📈 第三章：监督学习 —— 回归与分类

什么是监督学习？

监督学习就像有老师批改作业的学习方式：你做题（模型预测），老师告诉你对错（损失函数），你根据反馈调整（梯度下降）。

关键特点：训练数据有标签 —— 每个输入都对应一个正确的输出。

给定房子的特征，预测它的价格：

当 $\mathbf{X}^T\mathbf{X}$ 时可以直接求解。但在特征维度高或样本量大时，计算逆矩阵代价高，此时使用梯度下降更高效。

梯度下降优化 —— 完整Python实现：

import numpy as np

class LinearRegressionGD:
    """使用梯度下降的线性回归"""
    def __init__(self, learning_rate=0.01, n_iterations=1000):
        self.lr = learning_rate
        self.n_iter = n_iterations
        self.weights = None
        self.bias = None
        self.losses = []

    def fit(self, X, y):
        n_samples, n_features = X.shape
        self.weights = np.zeros(n_features)
        self.bias = 0

        for _ in range(self.n_iter):
            y_pred = X.dot(self.weights) + self.bias
            loss = np.mean((y_pred - y) ** 2)
            self.losses.append(loss)

            dw = (1 / n_samples) * (2 * X.T.dot(y_pred - y))
            db = (1 / n_samples) * (2 * np.sum(y_pred - y))

            self.weights -= self.lr * dw
            self.bias -= self.lr * db

    def predict(self, X):
        return X.dot(self.weights) + self.bias

正则化回归

正则化通过在损失函数中添加惩罚项，防止模型过拟合：

方法正则化项特点适用场景
Ridge (L2)权重趋向于小值，但不会精确为零特征间存在多重共线性
Lasso (L1)可以将部分权重压缩为零，实现特征选择高维稀疏数据
ElasticNet结合L1和L2的优点特征多且存在相关性
 
from sklearn.linear_model import Ridge, Lasso, ElasticNet

ridge = Ridge(alpha=1.0)      # L2正则化
lasso = Lasso(alpha=0.1)      # L1正则化
elastic = ElasticNet(alpha=0.1, l1_ratio=0.5)  # 混合正则化

ridge.fit(X_train, y_train)
lasso.fit(X_train, y_train)
elastic.fit(X_train, y_train)

print(f"Ridge R²:   {ridge.score(X_test, y_test):.4f}")
print(f"Lasso R²:   {lasso.score(X_test, y_test):.4f}")
print(f"ElasticNet: {elastic.score(X_test, y_test):.4f}")

多项式回归

当数据的非线性关系明显时，多项式回归通过添加高次特征来拟合曲线：

from sklearn.preprocessing import PolynomialFeatures
from sklearn.linear_model import LinearRegression
from sklearn.pipeline import make_pipeline

# 生成非线性数据
np.random.seed(42)
X = np.sort(np.random.rand(100, 1) * 10, axis=0)
y = np.sin(X).ravel() + np.random.randn(100) * 0.3

# 多项式回归：degree控制多项式阶数
model = make_pipeline(
    PolynomialFeatures(degree=3),
    LinearRegression()
)
model.fit(X, y)
print(f"多项式回归 R²: {model.score(X, y):.4f}")

过拟合与欠拟合

问题表现原因解决方案
欠拟合训练集和测试集表现都差模型太简单，无法捕捉数据规律增加模型复杂度、添加特征、减少正则化
过拟合训练集表现好，测试集表现差模型太复杂，记住了噪声增加数据、正则化、Dropout、早停
良好拟合训练集和测试集表现都较好模型复杂度适中保持当前配置

偏差-方差权衡（Bias-Variance Tradeoff）

模型的泛化误差可以分解为三个部分：

其中：

  • 偏差（Bias）：模型预测值与真实值的差距，反映模型的拟合能力。高偏差 = 欠拟合。
  • 方差（Variance）：模型在不同训练集上预测结果的波动程度，反映模型的稳定性。高方差 = 过拟合。
  • 不可约误差：数据本身的噪声，无法通过模型优化消除。

模型越简单 → 偏差高、方差低；模型越复杂 → 偏差低、方差高。好的模型需要在两者之间找到平衡点。
