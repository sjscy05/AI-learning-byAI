YOLO —— 实时目标检测

YOLO（You Only Look Once）把目标检测变成一个回归问题：输入一张图，直接输出所有物体的位置和类别。

YOLO的核心思想：

  1. 网格划分：将图像划分为 $S \times S$ 的网格
  2. 边界框预测：每个网格预测 $B$ 个边界框（每个框包含 $x, y, w, h$ 和置信度）
  3. 类别预测：每个网格预测 $C$ 个类别的概率

其中 $t_x, t_y$ 是偏移量，$b_x, b_y$ 是中心坐标，$p_w, p_h$ 是锚框尺寸。

NMS（非极大值抑制）：去除重叠的检测框，只保留置信度最高的：

  1. 按置信度排序所有检测框
  2. 选择置信度最高的框
  3. 计算与其他框的IoU，移除IoU>阈值的框
  4. 重复直到没有框被移除
# 使用YOLO进行目标检测
from ultralytics import YOLO

# 加载预训练模型
model = YOLO('yolov8n.pt')  # nano版本，速度最快

# 检测图像中的物体
results = model('street.jpg')

# 查看检测结果
for result in results:
    boxes = result.boxes  # 边界框
    for box in boxes:
        # 获取坐标、置信度和类别
        x1, y1, x2, y2 = box.xyxy[0]
        confidence = box.conf[0]
        class_id = box.cls[0]
        class_name = result.names[int(class_id)]

        print(f"检测到: {class_name}, "
              f"置信度: {confidence:.2f}, "
              f"位置: ({x1:.0f},{y1:.0f})-({x2:.0f},{y2:.0f})")

Vision Transformer（ViT）—— Transformer做视觉

核心思想：将图像分成固定大小的块（patch），每个块当作一个"词"，用Transformer处理。

图像 → 切块 → 展平 → 加位置编码 → Transformer编码器 → 分类

224×224图像 → 16×16=196个patch → 每个patch是一个token

import torch
import torch.nn as nn

class PatchEmbedding(nn.Module):
    """将图像切分为patch并嵌入"""
    def __init__(self, img_size=224, patch_size=16, in_channels=3, embed_dim=768):
        super().__init__()
        self.patch_size = patch_size
        self.n_patches = (img_size // patch_size) ** 2  # 196

        # 用卷积实现切块+嵌入
        self.proj = nn.Conv2d(in_channels, embed_dim,
                              kernel_size=patch_size, stride=patch_size)

    def forward(self, x):
        # x: (batch, 3, 224, 224)
        x = self.proj(x)         # (batch, 768, 14, 14)
        x = x.flatten(2).transpose(1, 2)  # (batch, 196, 768)
        return x

class VisionTransformer(nn.Module):
    def __init__(self, img_size=224, patch_size=16, n_classes=1000,
                 embed_dim=768, depth=12, n_heads=12):
        super().__init__()
        self.patch_embed = PatchEmbedding(img_size, patch_size, embed_dim=embed_dim)

        # CLS token和位置编码
        self.cls_token = nn.Parameter(torch.randn(1, 1, embed_dim))
        self.pos_embed = nn.Parameter(torch.randn(1, self.patch_embed.n_patches + 1, embed_dim))

        # Transformer编码器
        encoder_layer = nn.TransformerEncoderLayer(
            d_model=embed_dim, nhead=n_heads, dim_feedforward=embed_dim*4
        )
        self.transformer = nn.TransformerEncoder(encoder_layer, num_layers=depth)

        # 分类头
        self.head = nn.Linear(embed_dim, n_classes)

    def forward(self, x):
        batch_size = x.shape[0]
        x = self.patch_embed(x)  # (batch, 196, 768)

        # 添加CLS token
        cls_tokens = self.cls_token.expand(batch_size, -1, -1)
        x = torch.cat([cls_tokens, x], dim=1)  # (batch, 197, 768)

        # 添加位置编码
        x = x + self.pos_embed

        # Transformer编码
        x = x.transpose(0, 1)  # (seq_len, batch, embed_dim)
        x = self.transformer(x)

        # 用CLS token做分类
        x = x[0]  # 取CLS token
        x = self.head(x)
        return x

# ViT-B/16: 86M参数, patch_size=16, embed_dim=768, depth=12
model = VisionTransformer(img_size=224, patch_size=16, n_classes=1000)
print(f"ViT-B/16参数量: {sum(p.numel() for p in model.parameters()):,}")

练习题

练习1：实现一个简单的GAN，在MNIST数据集上生成手写数字。观察训练过程中生成图像的变化。

💡 指引： GAN的训练是一个博弈过程：生成器试图"骗过"判别器，判别器试图区分真假。注意训练平衡——如果判别器太强，生成器梯度消失；如果太弱，生成器学不到有意义的东西。建议先从简单架构开始，观察loss曲线的波动。

练习2：解释GAN训练中的模式崩塌（mode collapse）问题。提出至少两种缓解方法。

💡 指引： 模式崩塌是指生成器只学会生成少数几种样本（"偷懒"），而忽略数据的多样性。常用缓解方法：1）mini-batch discrimination让判别器看到一批样本而非单个；2）unrolled GAN让生成器考虑判别器的未来更新；3）使用WGAN等替代损失函数。

练习3：使用YOLOv8在自定义数据集上训练目标检测模型，评估mAP指标。

💡 指引： YOLO的核心思想是将检测转化为回归问题——一次性预测所有边界框和类别。mAP（mean Average Precision）综合考虑了检测的准确率和召回率。数据标注质量对结果影响很大，注意检查标注格式是否符合YOLO要求。

练习4：比较CNN和ViT在小数据集（如CIFAR-10）和大数据集（如ImageNet-21k）上的表现差异。

💡 指引： CNN有归纳偏置（局部性、平移不变性），在小数据集上泛化更好；ViT几乎没有视觉先验，需要大量数据才能学到类似的空间关系。但在超大数据集上，ViT的全局注意力机制能捕获CNN难以学到的长距离依赖。
