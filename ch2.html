🍉 第二章：数据与特征

数据的基本类型

在开始特征工程之前，先了解数据的基本分类方式。不同类型的数据需要不同的处理方法。

按数据结构分类

类型特征示例
结构化数据行列分明，每行代表一条记录，每列代表一个特征数据库表、Excel表格、CSV文件
非结构化数据没有固定格式，无法直接用表格表示文本、图片、音频、视频
半结构化数据有一定的结构但不如结构化数据严格JSON、XML、HTML、日志文件

按特征类型分类

类型子类型示例处理方法
数值型连续型身高 175.5 cm、温度 36.5°C归一化、标准化
数值型离散型家庭人数 3、购买次数 5可直接使用或分箱
类别型有序型学历：小学 < 初中 < 高中 < 大学有序编码
类别型无序型颜色：红、绿、蓝独热编码
文本型-评论、新闻、邮件TF-IDF、词嵌入
时间型-日期、时间戳、时间段提取年月日、时间差

西瓜问题的完整特征工程

让我们用一个完整的例子来演示特征工程的全过程。

原始数据表格

假设我们从水果摊收集了以下数据：

编号颜色纹路瓜蒂敲声重量(kg)大小(cm)含糖量(%)好瓜
1翠绿清晰卷曲清脆4.22214.5是
2深绿模糊直沉闷3.1189.2否
3浅绿清晰卷曲清脆5.02516.8是
4翠绿无脱落咚咚2.8167.5否

类别型特征编码

机器学习模型无法直接理解"翠绿"、"清脆"这样的文字，需要转换为数字。

① 独热编码（One-Hot Encoding） —— 适用于无序类别：

import pandas as pd
from sklearn.preprocessing import OneHotEncoder

# 独热编码示例
colors = pd.DataFrame({'颜色': ['翠绿', '深绿', '浅绿', '翠绿', '深绿']})

encoder = OneHotEncoder(sparse_output=False)
colors_encoded = encoder.fit_transform(colors)
print(colors_encoded)
# 输出: 
# [[1, 0, 0],   # 翠绿 → [1,0,0]
#  [0, 1, 0],   # 深绿 → [0,1,0]
#  [0, 0, 1],   # 浅绿 → [0,0,1]
#  [1, 0, 0],   # 翠绿 → [1,0,0]
#  [0, 1, 0]]   # 深绿 → [0,1,0]

② 有序编码（Ordinal Encoding） —— 适用于有顺序的类别：

from sklearn.preprocessing import OrdinalEncoder

# 纹路: 有自然顺序 (无 < 模糊 < 清晰)
纹路顺序 = [['无', '模糊', '清晰']]
encoder = OrdinalEncoder(categories=纹路顺序)
纹路编码 = encoder.fit_transform(df[['纹路']])

数值型特征处理

重量（4.2kg）和含糖量（14.5%）量纲不同，需要缩放到相同范围。

from sklearn.preprocessing import MinMaxScaler, StandardScaler

# 原始数值特征
weight = [4.2, 3.1, 5.0, 2.8, 4.5]
sugar = [14.5, 9.2, 16.8, 7.5, 15.3]

# 方法1: Min-Max归一化 → 缩放到 [0, 1]
scaler = MinMaxScaler()
weight_minmax = scaler.fit_transform([[4.2],[3.1],[5.0],[2.8],[4.5]])
# 输出: [0.692, 0.269, 1.0, 0.0, 0.808]

# 方法2: Z-score标准化 → 均值0, 标准差1
scaler = StandardScaler()
weight_zscore = scaler.fit_transform([[4.2],[3.1],[5.0],[2.8],[4.5]])
# 输出: [0.216, -0.862, 1.203, -1.382, 0.479]
方法公式适用场景特点
Min-Max归一化(x - min) / (max - min)数据分布有明确边界输出在[0,1]，对异常值敏感
Z-score标准化(x - mean) / std数据近似正态分布输出均值0方差1，对异常值鲁棒

特征工程的完整流程

一个完整的特征工程流程包含四个核心步骤：

步骤1：数据清洗

数据质量决定模型效果的天花板。

import pandas as pd
import numpy as np

df = pd.read_csv('watermelon.csv')

# 缺失值处理
df.fillna(df.mean(), inplace=True)          # 数值型: 用均值填充
df['颜色'].fillna(df['颜色'].mode()[0])      # 类别型: 用众数填充

# 异常值检测 (IQR方法)
Q1 = df['重量'].quantile(0.25)
Q3 = df['重量'].quantile(0.75)
IQR = Q3 - Q1
df = df[(df['重量'] >= Q1-1.5*IQR) & (df['重量'] <= Q3+1.5*IQR)]

# 重复值删除
df.drop_duplicates(inplace=True)

步骤2：特征构造

从已有特征中创造更有信息量的新特征。

# 组合特征: 重量与大小的比值反映密度
df['密度'] = df['重量'] / (df['大小'] ** 3)

# 多项式特征: 捕捉非线性关系
from sklearn.preprocessing import PolynomialFeatures
poly = PolynomialFeatures(degree=2)
X_poly = poly.fit_transform(df[['重量', '大小']])

# 时间特征: 从日期中提取信息
df['购买日期'] = pd.to_datetime(df['购买日期'])
df['星期'] = df['购买日期'].dt.dayofweek
df['是否周末'] = df['星期'].isin([5, 6]).astype(int)

步骤3：特征选择

从众多特征中选出最有价值的子集。

方法原理优点适用场景
过滤法根据统计指标（相关系数、卡方检验）排序计算快、与模型无关高维数据快速筛选
包裹法用模型性能评估特征子集针对特定模型优化特征数量适中时
嵌入法模型训练中自动选择效率高、效果好树模型、正则化模型
 
# 过滤法: 相关系数
correlations = df.corr()['好瓜'].abs().sort_values(ascending=False)

# 嵌入法: L1正则化自动选择
from sklearn.linear_model import LassoCV
model = LassoCV()
model.fit(X, y)
importance = np.abs(model.coef_)
selected = X.columns[importance > 0]

步骤4：特征降维

当特征维度太高时，降维可以减少计算量、去除噪声。

# PCA: 保留95%方差
from sklearn.decomposition import PCA
pca = PCA(n_components=0.95)
X_reduced = pca.fit_transform(X)

# t-SNE: 可视化降维（保留局部结构）
from sklearn.manifold import TSNE
X_2d = TSNE(n_components=2).fit_transform(X)
方法特点适用场景
PCA线性降维，保留全局方差，可逆高维数值数据压缩
t-SNE非线性降维，保留局部结构，不可逆数据可视化
UMAP比t-SNE更快，更好保留全局结构大规模数据可视化

特征工程的最佳实践

"数据和特征决定了机器学习的上限，而模型和算法只是逼近这个上限。" —— Andrew Ng

一个简单的模型 + 好的特征，往往比复杂模型 + 差的特征效果更好。以下是5条经过验证的最佳实践：

  • 先理解业务再动手 —— 特征工程不是盲目尝试，先理解数据背后的业务逻辑
  • 数据清洗优先 —— 垃圾进垃圾出，确保数据质量后再做特征构造
  • 从简单开始 —— 先用基础特征建基线模型，再逐步增加复杂度
  • 避免数据泄露 —— 特征缩放、缺失值填充等操作必须在训练集上fit，再应用到测试集
  • 特征可解释性 —— 可解释的特征更易调试、更值得信任
