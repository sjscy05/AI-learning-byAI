人工神经元的数学模型

生物神经元通过树突接收信号、轴突传递信号。人工神经元模仿了这个过程，将输入信号加权求和后，通过激活函数产生输出。

激活：将加权求和的结果通过一个非线性函数 $f$ 得到输出：

符号含义维度
$\mathbf{x}$输入向量$n \times 1$（n个特征）
$\mathbf{w}$权重向量$n \times 1$（每个输入一个权重）
$b$偏置（bias）标量
$f(\cdot)$激活函数非线性映射
$a$输出（activation）标量
为什么需要激活函数？如果没有激活函数，无论网络有多少层，整个网络等价于一个线性变换，无法学习复杂的非线性模式。激活函数赋予了神经网络"学习万物"的能力。

激活函数详解

激活函数是神经网络的灵魂。不同的激活函数有不同的数学特性和适用场景：

1. Sigmoid 函数

  • 输出范围：$(0, 1)$，可以解释为概率
  • 优点：输出有界，适合二分类的输出层
  • 缺点：输出非零中心（均值≈0.5），导致梯度更新效率低；深度网络中容易出现梯度消失（梯度最大值仅 0.25）

2. Tanh 函数

  • 输出范围：$(-1, 1)$，零中心化
  • 优点：输出以0为中心，比Sigmoid收敛更快
  • 缺点：同样存在梯度消失问题（但比Sigmoid轻）

3. ReLU 函数

  • 输出范围：$[0, +\infty)$
  • 优点：计算极快、缓解梯度消失、产生稀疏激活
  • 缺点：Dead ReLU问题 —— 当输入为负时梯度为0，神经元"永久死亡"，不再更新

4. Leaky ReLU 函数

其中 $\alpha$ 是一个小常数（如0.01）。当输入为负时仍保留微小梯度，有效缓解Dead ReLU问题。

5. Softmax 函数

  • 输出范围：$(0, 1)$，所有输出之和为1
  • 适用场景：多分类问题的输出层，将logits转换为概率分布

激活函数对比：

激活函数输出范围零中心梯度消失计算开销适用场景
Sigmoid$(0, 1)$否严重中二分类输出层
Tanh$(-1, 1)$是中等中RNN/LSTM
ReLU$[0, +\infty)$近似无极低隐藏层首选
Leaky ReLU$(-\infty, +\infty)$近似无极低避免Dead ReLU
Softmax$(0, 1)$否不适用高多分类输出层

反向传播算法详解

反向传播是神经网络学习的核心算法。它的本质是微积分中的链式法则（Chain Rule）在计算图上的高效应用。

链式法则

如果 $y = f(g(x))$，则 $\frac{dy}{dx} = \frac{dy}{dg} \cdot \frac{dg}{dx}$。反向传播就是从输出层开始，沿着计算图反向逐层应用链式法则，计算损失函数对每个参数的梯度。

前向传播示例（5步）

以一个简单的两层网络为例，输入 $x=2$，目标输出 $y=1$：

步骤计算结果
① 输入层$x = 2$$a^{(0)} = 2$
② 隐藏层加权求和$z^{(1)} = w_1 \cdot a^{(0)} + b_1 = 0.5 \times 2 + 0.1$$z^{(1)} = 1.1$
③ 隐藏层激活$a^{(1)} = \text{ReLU}(1.1)$$a^{(1)} = 1.1$
④ 输出层加权求和$z^{(2)} = w_2 \cdot a^{(1)} + b_2 = 0.3 \times 1.1 + 0.05$$z^{(2)} = 0.38$
⑤ 输出层激活$\hat{y} = \text{ReLU}(0.38)$$\hat{y} = 0.38$

反向传播梯度计算（8步）

损失函数 $L = \frac{1}{2}(y - \hat{y})^2 = \frac{1}{2}(1 - 0.38)^2 = 0.1922$

步骤计算结果
① $\frac{\partial L}{\partial \hat{y}}$$\hat{y} - y$$-0.62$
② $\frac{\partial \hat{y}}{\partial z^{(2)}}$$\text{ReLU}'(0.38)$$1$
③ $\frac{\partial L}{\partial w_2}$$\frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial z^{(2)}} \cdot a^{(1)}$$-0.682$
④ $\frac{\partial L}{\partial b_2}$$\frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial z^{(2)}}$$-0.62$
⑤ $\frac{\partial L}{\partial a^{(1)}}$$\frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial z^{(2)}} \cdot w_2$$-0.186$
⑥ $\frac{\partial L}{\partial z^{(1)}}$$\frac{\partial L}{\partial a^{(1)}} \cdot \text{ReLU}'(1.1)$$-0.186$
⑦ $\frac{\partial L}{\partial w_1}$$\frac{\partial L}{\partial z^{(1)}} \cdot a^{(0)}$$-0.372$
⑧ $\frac{\partial L}{\partial b_1}$$\frac{\partial L}{\partial z^{(1)}}$$-0.186$

得到所有梯度后，使用梯度下降更新参数：$w \leftarrow w - \alpha \frac{\partial L}{\partial w}$。

完整Python实现（NeuralNetwork类）

import numpy as np

class NeuralNetwork:
    """一个完整的两层神经网络，含前向传播和反向传播"""
    def __init__(self, input_size, hidden_size, output_size, lr=0.01):
        self.lr = lr
        # He初始化
        self.W1 = np.random.randn(input_size, hidden_size) * np.sqrt(2 / input_size)
        self.b1 = np.zeros((1, hidden_size))
        self.W2 = np.random.randn(hidden_size, output_size) * np.sqrt(2 / hidden_size)
        self.b2 = np.zeros((1, output_size))

    def relu(self, z):
        return np.maximum(0, z)

    def relu_derivative(self, z):
        return (z > 0).astype(float)

    def forward(self, X):
        """前向传播"""
        self.z1 = X.dot(self.W1) + self.b1
        self.a1 = self.relu(self.z1)
        self.z2 = self.a1.dot(self.W2) + self.b2
        self.a2 = self.z2  # 回归任务直接输出
        return self.a2

    def backward(self, X, y):
        """反向传播，计算梯度并更新参数"""
        m = X.shape[0]

        # 输出层梯度
        dz2 = self.a2 - y
        dW2 = (1 / m) * self.a1.T.dot(dz2)
        db2 = (1 / m) * np.sum(dz2, axis=0, keepdims=True)

        # 隐藏层梯度
        dz1 = dz2.dot(self.W2.T) * self.relu_derivative(self.z1)
        dW1 = (1 / m) * X.T.dot(dz1)
        db1 = (1 / m) * np.sum(dz1, axis=0, keepdims=True)

        # 梯度下降更新
        self.W2 -= self.lr * dW2
        self.b2 -= self.lr * db2
        self.W1 -= self.lr * dW1
        self.b1 -= self.lr * db1

    def train(self, X, y, epochs=1000):
        for epoch in range(epochs):
            pred = self.forward(X)
            self.backward(X, y)
            if epoch % 200 == 0:
                loss = np.mean((pred - y) ** 2)
                print(f"Epoch {epoch}, Loss: {loss:.6f}")

# 使用示例
X = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
y = np.array([[0], [1], [1], [0]])  # XOR问题

nn.train(X, y, epochs=2000)
print("预测结果:", nn.forward(X).round())

常见优化器对比

梯度下降的"步长"和"方向"策略决定了训练效率。不同优化器的核心区别在于如何利用历史梯度信息来加速收敛：

优化器核心思想学习率适用场景收敛速度
SGD最朴素的梯度下降固定或衰减凸优化、简单模型慢
Momentum引入"惯性"，加速通过平坦区域固定有鞍点的损失面较快
Adam自适应学习率 + 动量自适应通用首选快
AdamWAdam + 解耦权重衰减自适应大模型训练快且泛化好

正则化技术

正则化用于防止过拟合 —— 模型在训练集上表现好但在测试集上表现差。以下是三种最常用的正则化技术：

1. Dropout —— 随机丢弃神经元

训练时以概率 $p$ 随机将部分神经元输出置零，迫使网络学习冗余表示，避免对单个神经元的过度依赖。推理时不使用Dropout。

import torch.nn as nn

class MyModel(nn.Module):
    def __init__(self):
        super().__init__()
        self.network = nn.Sequential(
            nn.Linear(784, 256),
            nn.ReLU(),
            nn.Dropout(p=0.5),       # 50%的神经元随机置零
            nn.Linear(256, 128),
            nn.ReLU(),
            nn.Dropout(p=0.3),
            nn.Linear(128, 10)
        )

    def forward(self, x):
        return self.network(x)

# 训练时开启Dropout
model.train()
output = model(x)

# 推理时关闭Dropout
model.eval()
output = model(x)

2. Batch Normalization —— 批归一化

对每个mini-batch的激活值进行归一化（均值0、方差1），然后通过可学习的参数 $\gamma$ 和 $\beta$ 恢复表达能力。好处：加速训练、允许使用更大学习率、自带轻微正则化效果。

class MyModel(nn.Module):
    def __init__(self):
        super().__init__()
        self.network = nn.Sequential(
            nn.Linear(784, 256),
            nn.BatchNorm1d(256),   # 批归一化
            nn.ReLU(),
            nn.Linear(256, 128),
            nn.BatchNorm1d(128),
            nn.ReLU(),
            nn.Linear(128, 10)
        )

    def forward(self, x):
        return self.network(x)

# 注意: BatchNorm在训练和推理时行为不同
model.train()  # 使用batch统计量
model.eval()   # 使用全局统计量

3. 数据增强 —— 增加训练数据多样性

通过对训练数据施加随机变换（旋转、翻转、裁剪、颜色抖动等），人为增加数据多样性，有效防止过拟合。

from torchvision import transforms

# 训练时的数据增强
train_transform = transforms.Compose([
    transforms.RandomHorizontalFlip(p=0.5),       # 随机水平翻转
    transforms.RandomRotation(degrees=15),          # 随机旋转±15°
    transforms.RandomCrop(32, padding=4),           # 随机裁剪
    transforms.ColorJitter(brightness=0.2,          # 颜色抖动
                           contrast=0.2,
                           saturation=0.2),
    transforms.ToTensor(),
    transforms.Normalize((0.5,), (0.5,))
])

# 验证时不做增强
val_transform = transforms.Compose([
    transforms.ToTensor(),
    transforms.Normalize((0.5,), (0.5,))
])

from torchvision.datasets import CIFAR10
train_dataset = CIFAR10(root='./data', train=True,
                        transform=train_transform, download=True)
val_dataset = CIFAR10(root='./data', train=False,
                      transform=val_transform)
