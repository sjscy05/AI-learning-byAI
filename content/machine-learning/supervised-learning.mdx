# 监督学习

## 什么是监督学习

监督学习是机器学习的一种方法，其中模型从标记的训练数据中学习。

## 线性回归

线性回归是最简单的监督学习算法之一：

$$y = \mathbf{w}^T \mathbf{x} + b$$

其中：
- $\mathbf{w}$ 是权重向量
- $\mathbf{x}$ 是输入特征
- $b$ 是偏置项

### 损失函数

我们使用均方误差（MSE）作为损失函数：

$$L(\mathbf{w}, b) = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2$$

### 梯度下降

为了最小化损失函数，我们使用梯度下降：

$$\mathbf{w} \leftarrow \mathbf{w} - \alpha \nabla L(\mathbf{w})$$

其中 $\alpha$ 是学习率。
