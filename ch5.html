🖼️ 第五章：卷积神经网络与图像识别

滑动的小窗口找猫耳朵 —— 卷积是什么？

想象你在一张巨大的照片中找猫耳朵。你不会一次看整张照片，而是拿着一个小方框（比如3×3像素），在照片上从左到右、从上到下滑动，每次只看一小块区域。

这就是卷积操作 —— 用一个小的"滤波器"（卷积核）在图像上滑动，提取局部特征。

卷积操作的伪代码

# 卷积操作伪代码
def conv2d(image, kernel):
    """
    image: 输入图像 (H x W)
    kernel: 卷积核 (K x K)
    """
    H, W = image.shape
    K = kernel.shape[0]
    output = zeros((H-K+1, W-K+1))

    # 滑动窗口
    for i in range(H - K + 1):
        for j in range(W - K + 1):
            # 取出当前窗口
            window = image[i:i+K, j:j+K]
            # 逐元素相乘再求和
            output[i, j] = sum(window * kernel)

    return output

# 常见卷积核示例
# 边缘检测核
edge_kernel = [[-1, -1, -1],
               [-1,  8, -1],
               [-1, -1, -1]]

# 模糊核
blur_kernel = [[1, 1, 1],
               [1, 1, 1],
               [1, 1, 1]] / 9

常见卷积类型

1. 标准卷积

每个卷积核同时在空间维度（宽×高）和通道维度上进行卷积。假设输入 Cin 个通道，输出 Cout 个通道：

参数量 = Cin × K × K × Cout + Cout (偏置)

2. 1×1 卷积

卷积核大小为 1×1，不改变空间尺寸，只调整通道数。常用于：

  • 通道数升降维：如从 256 通道压缩到 64 通道
  • 跨通道信息融合：相当于对每个像素做全连接
  • 增加非线性：配合激活函数引入非线性变换
# 1x1卷积：通道数调整
nn.Conv2d(256, 64, kernel_size=1)  # 256 → 64 通道
# 参数量: 256 × 1 × 1 × 64 + 64 = 16,448

3. 深度可分离卷积 (Depthwise Separable Convolution)

将标准卷积分解为两步：逐通道卷积 + 1×1卷积，大幅减少参数量：

类型参数量计算量适用场景
标准卷积Cin × K × K × CoutO(K²·Cin·Cout·H·W)精度优先
深度可分离Cin×K×K + Cin×CoutO(K²·Cin·H·W + Cin·Cout·H·W)移动端/轻量化
 
# 深度可分离卷积（MobileNet风格）
# 第一步：逐通道卷积（每个通道独立卷积）
depthwise = nn.Conv2d(64, 64, kernel_size=3, padding=1, groups=64)
# 第二步：1x1卷积（通道间信息融合）
pointwise = nn.Conv2d(64, 128, kernel_size=1)

# 参数量对比（输入64通道，输出128通道，3x3卷积核）：
# 标准卷积: 64 × 3 × 3 × 128 + 128 = 73,856
# 深度可分离: 64 × 3 × 3 + 64 × 128 + 128 = 8,896 (减少约88%)

1. LeNet-5 (1998) —— 第一个成功的CNN

  • 约 6万参数，仅用于灰度图像
  • 证明了CNN在图像识别上的可行性

2. AlexNet (2012) —— 深度学习里程碑

Alex Krizhevsky赢得ImageNet竞赛，将错误率从26%降到16%，引爆深度学习革命。

  • ReLU激活：替代Sigmoid，训练速度提升6倍
  • Dropout：防止过拟合
  • GPU训练：首次大规模使用GPU加速
  • 数据增强：翻转、裁剪、颜色变换
  • 6000万参数，5层卷积 + 3层全连接

3. VGGNet (2014) —— 3×3卷积的胜利

牛津大学提出，核心思想：用多个3×3小卷积核替代大卷积核。

这意味着：两个3×3卷积的感受野等于一个5×5卷积，但参数更少、非线性更强。

  • VGG-16：16层，1.38亿参数
  • VGG-19：19层，1.44亿参数
  • 结构简洁统一，非常适合迁移学习

4. ResNet (2015) —— 残差连接，突破深度限制

何恺明提出，解决了深层网络的梯度消失问题，使训练上百层网络成为可能。

残差学习的核心公式：

网络不再学习完整的映射 H(x)，而是学习残差 F(x) = H(x) - x。通过跳跃连接（skip connection），即使某一层学不到有用信息，至少能恒等映射。

class ResidualBlock(nn.Module):
    def __init__(self, channels):
        super().__init__()
        self.conv1 = nn.Conv2d(channels, channels, 3, padding=1)
        self.conv2 = nn.Conv2d(channels, channels, 3, padding=1)
        self.relu = nn.ReLU()

    def forward(self, x):
        residual = x                # 保存输入
        out = self.relu(self.conv1(x))
        out = self.conv2(out)
        out = out + residual        # 残差连接：输出 = F(x) + x
        return self.relu(out)
  • ResNet-50/101/152：深度可达152层
  • 2015年ImageNet冠军，错误率3.57%（超越人类水平）
  • 至今仍是工业界最常用的骨干网络之一

迁移学习 —— 站在巨人的肩膀上

核心思想：不需要从头训练一个巨大的CNN。在大规模数据集（如ImageNet）上预训练好的模型，已经学会了通用的视觉特征（边缘、纹理、形状）。我们只需要微调最后几层，就能适应自己的任务。

迁移学习的三种策略：

策略做法适用场景
特征提取冻结所有预训练层，只训练新的分类头数据量小、任务相似
微调解冻部分层，用小学习率联合训练数据量适中
全量微调解冻所有层，用很大学习率训练数据量大、任务差异大
 
import torch
import torch.nn as nn
from torchvision import models

# 1. 加载预训练ResNet-18
model = models.resnet18(weights=models.ResNet18_Weights.DEFAULT)

# 2. 冻结所有层（特征提取模式）
for param in model.parameters():
    param.requires_grad = False

# 3. 替换最后的全连接层（适配自己的分类任务）
num_classes = 5  # 比如分5类
model.fc = nn.Linear(model.fc.in_features, num_classes)

# 4. 只有新的分类头会被训练
optimizer = torch.optim.Adam(model.fc.parameters(), lr=1e-3)

# 训练循环
for epoch in range(10):
    for images, labels in train_loader:
        outputs = model(images)
        loss = nn.CrossEntropyLoss()(outputs, labels)
        loss.backward()
        optimizer.step()
        optimizer.zero_grad()
实践建议：如果你的数据集与ImageNet差异不大（如医学X光片分类），冻结大部分层效果更好。如果差异很大（如卫星图像分类），需要解冻更多层进行微调。

池化与全连接

  • 池化层（Pooling）：缩小特征图尺寸，保留关键信息。比如取2×2区域的最大值
  • 全连接层（FC）：将提取的特征"拍平"，进行最终分类
import torch.nn as nn

class SimpleCNN(nn.Module):
    def __init__(self):
        super().__init__()
        self.features = nn.Sequential(
            # 卷积层: 3通道输入 → 16通道输出, 3x3卷积核
            nn.Conv2d(3, 16, kernel_size=3, padding=1),
            nn.ReLU(),
            nn.MaxPool2d(2),  # 池化: 尺寸减半

            nn.Conv2d(16, 32, kernel_size=3, padding=1),
            nn.ReLU(),
            nn.MaxPool2d(2),

            nn.Conv2d(32, 64, kernel_size=3, padding=1),
            nn.ReLU(),
            nn.AdaptiveAvgPool2d(1)  # 全局平均池化
        )
        self.classifier = nn.Linear(64, 10)  # 10分类

    def forward(self, x):
        x = self.features(x)
        x = x.view(x.size(0), -1)  # 拍平
        return self.classifier(x)
